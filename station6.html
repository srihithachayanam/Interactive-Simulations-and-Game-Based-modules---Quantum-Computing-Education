No-Cloning Theorem
Quantum states cannot be copied — this is the no-cloning theorem. That's why we use entanglement-based encoding instead of simple backups.
If you could copy a qubit, you could read multiple copies and beat the Heisenberg uncertainty principle. Nature forbids this. Instead, quantum error correction uses redundant entangled encoding — spreading information across multiple qubits so errors can be detected without ever measuring the data directly.

Code Distance Exceeded
Two simultaneous errors exceeded this code's distance. A distance-3 code can only correct 1 error. More physical qubits are needed.
The 3-qubit repetition code has code distance d=3. This means it can detect up to 2 errors, but only correct 1. When two qubits flip simultaneously, the syndrome pattern becomes ambiguous — any correction makes things worse, not better.

Real quantum computers use surface codes with hundreds of physical qubits to protect a single logical qubit, achieving distance high enough to correct multiple simultaneous errors.
